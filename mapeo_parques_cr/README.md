## Mapeo Parques de CR

Esta disposición está diseñada para facilitar el mapeo de parques y espacios recreativos en Costa Rica. Permite documentar mobiliario urbano, vegetación, infraestructura y elementos característicos de parques públicos.

### Botones disponibles

| Botón | Descripción |
|-------|-------------|
| **Poste de luz** | Marca la ubicación de postes de iluminación en el parque |
| **Edificios** | Identifica edificaciones dentro o cerca del parque |
| **Parada de bus** | Ubicación de paradas de transporte público cercanas |
| **Elementos de parque de niños** | Juegos infantiles, columpios, resbaladeros, etc. |
| **Bancas** | Bancas o asientos disponibles en el parque |
| **Mesas** | Mesas para picnic o reuniones |
| **Basureros** | Contenedores de basura ubicados en el área |
| **Bebedero (no funcional)** | Fuentes o bebederos que no están en funcionamiento |
| **Puesto de guarda** | Caseta o lugar de vigilancia del parque |
| **Árboles con arbustos** | Zonas con vegetación arbórea y arbustiva |
| **Árbol sin arbusto** | Árboles individuales sin vegetación acompañante |
| **Plantas** | Zonas con plantas ornamentales o jardinería |
| **Tipo de piso - Zacate** | Áreas con césped o zacate natural |
| **Tipo de piso - Ladrillo** | Superficies pavimentadas con ladrillo |
| **Tipo de piso - Concreto** | Superficies de concreto o cemento |

### Funciones adicionales

- **Grabación de voz**: Permite añadir notas de audio
- **Cámara**: Toma fotografías georreferenciadas
- **Nota de texto**: Añade observaciones escritas
